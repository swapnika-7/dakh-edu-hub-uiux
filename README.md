# Dakh Edu Hub - UI/UX Design Project

This project is a Learning Management System (LMS) designed for students to watch courses, take notes, and track their learning progress.

## Live Prototype
https://id-preview--f7730cf7-512f-4f0a-af83-287da26bee97.lovable.app/

## Features
- Student dashboard
- Course listing with filters
- Video player with notes
- Progress tracking system

## About
This project was created using AI-assisted tools and focuses on UI/UX design for modern learning platforms.
